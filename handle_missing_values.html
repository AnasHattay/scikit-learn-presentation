Handle Missing Values

Different Methods to Deal with Missing Data

Drop Missing Values

Remove rows or columns that contain missing data

Row 1: 5.2
Row 2: NaN
Row 3: 7.8
↓
Row 1: 5.2
Row 3: 7.8

Forward Fill

Fill missing values with the previous valid value

Row 1: 5.2
Row 2: NaN
Row 3: 7.8
↓
Row 1: 5.2
Row 2: 5.2
Row 3: 7.8

Mean Imputation

Replace missing values with the mean of the column

Row 1: 5.2
Row 2: NaN
Row 3: 7.8
↓ Mean = 6.5
Row 1: 5.2
Row 2: 6.5
Row 3: 7.8

Backward Fill

Fill missing values with the next valid value

Row 1: 5.2
Row 2: NaN
Row 3: 7.8
↓
Row 1: 5.2
Row 2: 7.8
Row 3: 7.8

Interpolation

Estimate missing values based on surrounding values

Row 1: 5.0
Row 2: NaN
Row 3: 9.0
↓ Linear
Row 1: 5.0
Row 2: 7.0
Row 3: 9.0

Fill with Constant

Replace missing values with a specific constant value

Row 1: 5.2
Row 2: NaN
Row 3: 7.8
↓ Fill = 0
Row 1: 5.2
Row 2: 0.0
Row 3: 7.8
