Statistics with NumPy

Basic Statistical Functions

np.mean()
Arithmetic mean
data = np.array([1, 2, 3, 4, 5]) mean = np.mean(data) # 3.0
np.median()
Median value
data = np.array([1, 2, 3, 4, 5]) median = np.median(data) # 3.0
np.std()
Standard deviation
data = np.array([1, 2, 3, 4, 5]) std = np.std(data) # 1.58
np.var()
Variance
data = np.array([1, 2, 3, 4, 5]) var = np.var(data) # 2.5
np.percentile()
Percentile values
data = np.array([1, 2, 3, 4, 5]) p25 = np.percentile(data, 25) # 2.0 p75 = np.percentile(data, 75) # 4.0
np.corrcoef()
Correlation coefficients
# 4 arrays: study_hours, test_scores, sleep_hours, stress_level study = np.array([8, 6, 10, 4]) scores = np.array([85, 75, 95, 60]) sleep = np.array([6, 8, 5, 9]) stress = np.array([3, 2, 4, 1]) # Calculate correlation matrix for all 4 variables corr_matrix = np.corrcoef([study, scores, sleep, stress]) print(corr_matrix)

Correlation Matrix Visualization

What is a Correlation Matrix?

A correlation matrix shows how strongly different variables are related to each other. Values range from -1 to +1:

  • +1.0: Perfect positive correlation (variables move together)
  • 0.0: No correlation (variables are independent)
  • -1.0: Perfect negative correlation (variables move opposite)
