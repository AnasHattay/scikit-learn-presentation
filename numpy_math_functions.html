Useful NumPy Math Functions

Basic Mathematical Functions

np.sum()
Sum of array elements
arr = np.array([1, 2, 3, 4]) result = np.sum(arr) # 10
np.mean()
Arithmetic mean
arr = np.array([1, 2, 3, 4]) result = np.mean(arr) # 2.5
np.std()
Standard deviation
arr = np.array([1, 2, 3, 4]) result = np.std(arr) # 1.29
np.var()
Variance
arr = np.array([1, 2, 3, 4]) result = np.var(arr) # 1.67
np.min() / np.max()
Minimum and maximum values
arr = np.array([1, 2, 3, 4]) min_val = np.min(arr) # 1 max_val = np.max(arr) # 4
np.argmin() / np.argmax()
Indices of min/max values
arr = np.array([1, 2, 3, 4]) min_idx = np.argmin(arr) # 0 max_idx = np.argmax(arr) # 3

Advanced Mathematical Functions

np.exp()
Exponential function
arr = np.array([0, 1, 2]) result = np.exp(arr) # [1, 2.72, 7.39]
np.log()
Natural logarithm
arr = np.array([1, 2.72, 7.39]) result = np.log(arr) # [0, 1, 2]
np.sqrt()
Square root
arr = np.array([1, 4, 9, 16]) result = np.sqrt(arr) # [1, 2, 3, 4]
np.power()
Power function
arr = np.array([1, 2, 3, 4]) result = np.power(arr, 2) # [1, 4, 9, 16]
np.abs()
Absolute value
arr = np.array([-1, -2, 3, -4]) result = np.abs(arr) # [1, 2, 3, 4]
np.round()
Round to nearest integer
arr = np.array([1.2, 2.7, 3.5]) result = np.round(arr) # [1, 3, 4]
