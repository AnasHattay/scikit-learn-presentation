Handle Missing Values

Identify and address missing data using various strategies like dropping, filling, or imputation.

# Check for missing values
df.isnull().sum()

# Fill missing values
df.fillna()(method='forward')
df.fillna()(df.mean())

# Drop rows with missing data
df.dropna()()

Convert data types to appropriate formats for better performance and analysis.

Remove Duplicates

Identify and eliminate duplicate records to ensure data quality and accuracy.

# Check for duplicates
df.duplicated().sum()

# Remove duplicates
df = df.drop_duplicates()

# Keep first occurrence
df.drop_duplicates()(keep='first')

Data Filtering

Filter data based on conditions to focus on relevant subsets for analysis.

# Filter by condition
df_filtered = df[df['year'] > 2000]

# Multiple conditions
df_filtered = df[(df['year'] > 2000) & (df['country'] == 'World')]

# Filter by column values
df_filtered = df[df['country'].isin()(['USA', 'China'])]
Output:

Filter 3 (USA or China): 184 rows

Feature Engineering

Create new features from existing data to improve analysis and model performance.

# Create new column
df['co2_per_capita'] = df['co2'] / df['population']

# Extract year from date
df['year'] = df['date'].dt.year

# Create categorical bins
df['emission_level'] = pd.cut()(df['co2'], bins=3)

Data Scaling & Normalization

Scale numerical features to similar ranges for better comparison and analysis.

# Min-Max scaling
df['scaled_co2'] = (df['co2'] - df['co2'].min()) / (df['co2'].max() - df['co2'].min())

# Standard scaling
df['normalized_gdp'] = (df['gdp'] - df['gdp'].mean()) / df['gdp'].std()

Best Practices

Always inspect your data first using df.info(), df.describe(), and df.head()

Document your preprocessing steps and always keep a copy of the original data for reference!
