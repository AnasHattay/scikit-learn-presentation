Decision Trees for CO2 Prediction

What is a Decision Tree?

A decision tree is a supervised learning algorithm that makes predictions by learning simple decision rules from data features. It creates a tree-like model of decisions, splitting data based on feature values to reach predictions.

How It Works
Recursively splits the data based on features that best separate the target values, creating a tree structure from root to leaves.
Decision Rules
Each internal node represents a "test" on a feature (e.g., GDP > 1000?), each branch represents the outcome, and leaves represent predictions.
Interpretability
Highly interpretable! You can visualize and understand exactly how the model makes decisions by following the tree paths.
Non-Linear
Can capture complex non-linear relationships between features and target, unlike linear regression.
