Visualizing Data Aggregation with CO2 Emissions Data

The GroupBy Operation

co2_by_year = df.groupby('year')['co2'].sum()

Step-by-Step Process

1
Group rows by the 'year' column
2
Select the 'co2' column from each group
3
Apply the 'sum' function to aggregate values
4
Return a new Series with year as index
