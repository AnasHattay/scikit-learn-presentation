Matplotlib Visualizations

Essential Plot Types for Data Analysis

Matplotlib provides a comprehensive library of plot types for different analytical needs. Each visualization serves a specific purpose in understanding your data patterns and relationships.

Histogram

Shows the distribution of a single variable by dividing data into bins and counting frequency.

plt.hist(data, bins=30, alpha=0.7)
plt.xlabel('CO2 Emissions')
plt.ylabel('Frequency')
Output:
Histogram created showing distribution of CO2 emissions data with 30 bins. X-axis labeled 'CO2 Emissions', Y-axis labeled 'Frequency'.
Use when: Understanding data distribution, identifying outliers, checking normality

Box Plot

Displays the distribution of data through quartiles, showing median, outliers, and spread. Each point represents a data value, with outliers clearly visible.

plt.boxplot([data1, data2, data3])
plt.xticks([1, 2, 3], ['USA', 'China', 'India'])
Output:
Boxplot created comparing distributions of three datasets. X-axis labeled with country names: USA, China, India.
Use when: Comparing distributions across groups, identifying outliers (points far from the main cluster), showing quartiles and median

Scatter Plot

Shows the relationship between two continuous variables by plotting points.

plt.scatter(x, y, alpha=0.6, s=50)
plt.xlabel('GDP')
plt.ylabel('CO2 Emissions')
Output:
Scatter plot created showing relationship between GDP and CO2 emissions. Points are semi-transparent (alpha=0.6) with size 50.
Use when: Exploring correlations, identifying patterns, detecting outliers

Correlation Heatmap

Visualizes correlation matrix using colors to show relationships between variables.

import seaborn as sns
corr_matrix = df.corr()
sns.heatmap(corr_matrix, annot=True)
Output:
Correlation heatmap created showing relationships between all numeric variables. Values displayed as annotations on the heatmap.
Use when: Analyzing multiple variable relationships, feature selection, multicollinearity

Bar Chart

Compares values across different categories using rectangular bars.

plt.bar(categories, values, color='steelblue')
plt.xticks(rotation=45)
plt.ylabel('Emissions (Mt)')
Output:
Bar chart created comparing categories with steelblue colored bars. X-axis labels rotated 45 degrees for better readability.
Use when: Comparing categories, ranking items, showing discrete data

Line Plot

Shows trends over time by connecting data points with lines.

plt.plot(years, emissions, linewidth=2, marker='o')
plt.xlabel('Year')
plt.ylabel('Global CO2 (Mt)')
Output:
Line plot created showing trend of global CO2 emissions over time. Line width 2 with circular markers at each data point.
Use when: Time series analysis, showing trends, comparing multiple series
