Evaluation Metrics

MAE, RMSE, and R² Score for Regression Models

The Three Essential Metrics

MAE
Mean Absolute Error
The average of the absolute differences between predicted and actual values. Simple and intuitive!
Formula
MAE = (1/n) Σ |yi - ŷi|
"On average, predictions are off by X units"
Lower is better (0 = perfect)
Same units as target variable
RMSE
Root Mean Squared Error
Square root of the average of squared differences. Penalizes large errors more heavily than MAE.
Formula
RMSE = √[(1/n) Σ (yi - ŷi)²]
Similar to MAE but more sensitive to outliers
Lower is better (0 = perfect)
Always ≥ MAE (equality only if all errors are equal)
R² Score
Coefficient of Determination
Proportion of variance in the target variable that is predictable from the features. The "goodness of fit"!
Formula
R² = 1 - (SSres / SStot)
SSres = Σ(yi - ŷi)²  |  SStot = Σ(yi - ȳ)²
Range: -∞ to 1.0
Higher is better (1.0 = perfect)
R² = 0.85 means "85% of variance explained"

Visual Example

• Blue dots: Actual values
• Red line: Perfect predictions (y = ŷ)
• Vertical lines: Prediction errors
• MAE: Average length of those lines
• RMSE: Like MAE but gives more weight to longer lines
• R²: How close the dots cluster around the red line
