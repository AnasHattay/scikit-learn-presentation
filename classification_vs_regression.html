Classification vs Regression

What Are They?

Regression
Predicting Continuous Values
Regression models predict numerical values that can take any value within a range. The output is continuous and can be measured on a scale.
Real-World Examples:
House Price Prediction: Predict price ($150,000, $275,500, $1.2M)
Temperature Forecasting: Predict temperature (15.3°C, 22.7°C, 30.1°C)
CO2 Emissions: Predict emissions (245.6 Mt, 1,234.8 Mt, 3,567.2 Mt)
Weight Estimation: Predict person's weight (65.3 kg, 78.9 kg)
Stock Prices: Predict closing price ($125.67, $543.21)
Classification
Predicting Categories
Classification models predict discrete categories or labels. The output belongs to a specific class or group from a predefined set of options.
Real-World Examples:
Email Filtering: Classify as "Spam" or "Not Spam"
Disease Diagnosis: Classify as "Healthy", "Sick", "Critical"
Image Recognition: Classify as "Cat", "Dog", "Bird", "Other"
Sentiment Analysis: Classify as "Positive", "Negative", "Neutral"
Customer Churn: Classify as "Will Leave" or "Will Stay"

Output Type

Regression Output
-∞ ... 0 ... 1523.7 ... ∞

Continuous numbers anywhere on the number line
Example: Predicting CO2 = 1,523.7 Mt

Classification Output
Class A
Class B
Class C

Discrete labels from a fixed set of categories
Example: Email is "Spam" or "Not Spam"
