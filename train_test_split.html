Important Parameters

test_size
What: Proportion of data for testing
Values: 0.1 to 0.3 (10% to 30%)
Default: 0.25 (25%)
Tip: Use 0.2 (20%) as standard
random_state
What: Seed for random number generator
Values: Any integer (e.g., 42)
Purpose: Reproducibility
Tip: Always set it for consistent results
shuffle
What: Whether to shuffle data before split
Values: True (default) or False
Purpose: Randomize distribution
Tip: Keep True unless data is time-series
stratify
What: Maintain class distribution
Values: Target array or None
Purpose: Balanced splits in classification
Tip: Use for imbalanced datasets
