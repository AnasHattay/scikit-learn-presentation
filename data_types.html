Types of Data

Structured vs Unstructured Data

Structured Data

Organized & Tabular Format

"Data that is organized in a predefined format, typically stored in rows and columns, making it easy to search, query, and analyze using traditional database tools."

Common Examples:

CSV Files Comma-separated values
SQL Databases Relational databases
Excel Spreadsheets Tabular data
JSON Structured format
Customer Data (CSV)
customer_id, name, email, age, city, purchase_amount
1001, John Smith, john@email.com, 28, New York, 150.50
1002, Sarah Johnson, sarah@email.com, 34, Los Angeles, 275.00
1003, Mike Wilson, mike@email.com, 45, Chicago, 89.99
1004, Emily Davis, emily@email.com, 29, Houston, 320.75

Unstructured Data

Free-form & Variable Format

"Data that doesn't follow a predefined model or format, making it more difficult to collect, process, and analyze using traditional methods. Requires special tools and techniques."

Common Examples:

Images Photos, screenshots
Videos MP4, AVI files
Audio MP3, WAV files
Text Documents PDFs, Word docs
Social Media Posts, tweets
Product Photo
JPEG, 1920x1080, 2.3MB
Contains: Product image, colors, shapes
